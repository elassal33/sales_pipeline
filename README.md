# Sales Data Pipeline (PySpark / Databricks)

## Overview
Scalable sales data pipeline built with PySpark on Databricks using the **Bronze → Silver → Gold** architecture. The pipeline processes sales data from multiple sources and is fully automated using Databricks Jobs.

## Architecture
- **Bronze**: Ingest raw sales data  
- **Silver**: Clean, standardize, and transform data  
- **Gold**: Aggregate data for analytics and BI  

## Tech Stack
PySpark · Delta Lake · Databricks · Databricks Jobs

## Outcome
Automated, production-style pipeline handling large sales datasets and delivering clean, analytics-ready tables for reporting and BI use cases.
